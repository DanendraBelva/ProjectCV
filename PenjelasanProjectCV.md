# ProjectCV
Project CV adalah project yang saya buat untuk menampilkan perkenalan diri, skill, dan prestasi saya. Website ini dibangun dengan Native HTMl5 dan CSS.
Untuk layouting, saya menggunakan CSS Flexbox agar strukturnya rapi dan fleksibel. Website ini juga sudah sepenuhnya responsif dengan Media Queries, tampilannya akan otomatis menyesuaikan diri di layar besar maupun kecil. Saya juga menambahkan elemen visual Glassmorphism dan transisi Smooth Scrolling agar interaksi pengguna terasa lebih halus dan modern.

Note (Saya biasanya melakukan Competitive Proggraming maaf jika ada kekurangan dalam pembuatan web)
